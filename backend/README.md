# Endpoints da API

## Vias

### Listar todas as vias
- **Endpoint:** `/vias`
- **Método:** `GET`

### Pegar informações detalhadas de uma via específica pelo ID
- **Endpoint:** `/vias/:id`
- **Método:** `GET`

### Pegar informações completas de uma via específica pelo ID (com JOIN nas tabelas relacionadas)
- **Endpoint:** `/vias/detailed/:id`
- **Método:** `GET`
